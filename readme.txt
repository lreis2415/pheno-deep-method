This is the implementation codes for the pheno-deep method for mapping rice paddy distribution.

"model.h5" is the trained model for the classification of rice paddy in Jin'an district, Lu'an city, Anhui Province. 

“rice-paddy-distribution-using-pheno-deep.tif” is the classification result using the pheno-deep method in the study area (Jin'an district, Lu'an city, Anhui Province).
